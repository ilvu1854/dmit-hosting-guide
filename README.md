## 怎么选机房和线路：按需求对号入座

这是买 DMIT 时最容易踩坑的环节——很多人看到"香港"两个字就觉得延迟低、速度快，但如果买了 HKG.T1，电信联通实际路由可能还要绕到日本，延迟反而不如洛杉矶 Pro。

机房和线路是两回事。地理位置近不等于延迟低，关键看线路走向。

**个人博客 / 跑脚本测试**：洛杉矶 T1 TINY，$6.90/月，便宜够用，不要在线路上纠结。

**面向国内用户建站，预算有限**：洛杉矶 LAX.EB TINY，$9.99/月，CMIN2 优化对联通移动用户体验不错，配合 8 折码长期用下来很划算。

**追求最稳定的国内访问体验，电信用户**：洛杉矶 LAX.Pro 系列，三网 CN2 GIA 双向优化，晚高峰延迟基本稳在 150-180ms，丢包率极低。

**极致低延迟，服务国内用户**：香港 HKG.Pro，延迟能压到 30ms 以内，就是价格贵，$39.90/月起。配合优惠码用才合算。

**需要日本 IP / 面向日本或东南亚用户**：东京 TYO.Pro，CN2 GIA + 9929 + CMI，亚太延迟表现优秀。TYO.T1 是更便宜的入门选项，但无中国大陆优化。

**需要高防护 / 游戏服务器**：洛杉矶 LAX.sPro 系列，带 5Tbps+ DDoS 防护，圣何塞 SJC.T1 也有 20Gbps 防护，都是应对攻击的方案。

## FAQ

**Q: DMIT 优惠码在哪里输入？**
A: 结算页面有"Promotion Code"输入框，把优惠码粘贴进去，点"Validate Code"验证。折扣确认生效后再提交订单，否则优惠码可能没有计入。

**Q: 月付能用优惠码吗？**
A: 多数折扣码需要季付或年付才能触发，月付套餐通常用不了大折扣。如果打算长期用，年付本身就比月付便宜，再叠上循环折扣码，差价非常可观。

**Q: Tier 1 系列适合国内用户吗？**
A: Tier 1 是国际线路，没有中国大陆专项优化，国内访问延迟会比 Pro 和 Eyeball 高，且可能绕路。如果主要用途是服务国内用户或科学上网，不建议选 T1。T1 更适合流量大、对带宽要求高、对大陆优化要求不那么严格的场景，或者纯粹需要一个便宜的国际节点。

**Q: CN2 GIA 和 CMIN2 有什么区别，怎么选？**
A: CN2 GIA 是电信顶级精品线路，三网（电信、联通、移动）回国都走 GIA，延迟低、丢包率极低，价格也最贵。CMIN2 是移动的高端线路，联通移动用户体验很好，电信用户体验比 GIA 略差一点，但价格比 Pro 系列低一档。电信用户首选 Pro，联通移动用户 Eyeball 系列性价比更高。

**Q: 买了 DMIT 之后 IP 被封怎么办？**
A: 每 15 天有一次免费换 IP 的机会，自助操作，不用开工单。其他情况换 IP 是 $5/次，遇到这种情况先看是不是 IP 被封，还是网络环境问题，再决定要不要换。

**Q: 优惠码有有效期吗？**
A: DMIT 的优惠码没有统一的到期时间，循环码本身是长期有效的，但官方可能在任何时候停用。建议在下单前进入结算页面验证一下，确认折扣到账再付款。
